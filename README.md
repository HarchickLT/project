Учебный проект по JavaScript.

Форма регистрации с клиентской валидацией данных.

Реализовано:
- обработка отправки формы
- предотвращение стандартного поведения формы
- получение данных из input
- проверка заполненности полей
- проверка совпадения паролей
- вывод сообщений об ошибках пользователю

Технологии:
HTML, JavaScript
